# 🚀 Antigram

**Antigram** is a powerful, open-source Telegram bot that acts as a remote control bridge to your local IDE and Browser automation agents (like Antigravity). 

It allows you to securely send prompts, upload files, read outputs, manage version control, and receive proactive task completion notifications—all from your phone via Telegram.

---

## ✨ Features

- **🗣️ Free Voice Notes to IDE Prompts**: Send a voice note in Telegram, and Antigram will seamlessly transcribe it locally using Google Speech infrastructure (bypassing OpenAI keys completely) and inject the prompt straight into your IDE agent.
- **📂 File Management on the Go**: Read active files with `/read`, list your active workspace directory with `/list`, or seamlessly download local dev server files to your phone with `/download`.
- **🛠️ Remote SSH-like Terminal**: Execute commands directly in your local environment via Telegram using `/terminal`. 
- **🌐 Proactive Completion Watcher**: If your IDE automation agent is executing a lengthy prompt, you can step away. Antigram watches the local DOM silently and sends a Telegram ping the moment the "Generating" state finishes!
- **🐙 Integrated Git Ops**: Fearlessly `/commit`, view `/diff`s, or securely `/push` code all generated by your local agents while you are away from your laptop.
- **📤 Drop & Deliver**: Send photos or documents to the Telegram bot, and it will effortlessly ingest them directly into your active workspace folder.

## 📦 Installation

### Prerequisites
- [Node.js](https://nodejs.org/en/) v16 or higher
- A Chromium-based browser or IDE that inherently exposes a Chrome DevTools Protocol (`CDP`) debugger port on `9222`.

### Step 1: Clone and Install
```bash
git clone https://github.com/your-username/antigram.git
cd antigram
npm install
```

### Step 2: Configure your Bot

**Option A — Interactive Setup Wizard (Recommended)**

Run the guided setup wizard and simply answer the prompts:
```bash
npm run setup
```

You will be asked step-by-step for:
- Your **Telegram Bot Token** (get this from [@BotFather](https://t.me/BotFather) → `/newbot`)
- Your **Telegram User ID** (get this from [@userinfobot](https://t.me/userinfobot))
- Optional advanced settings (port, IDE path, Wit.ai voice token)

All values are saved automatically to a `.env` file. To update them later, run:
```bash
npm run setup:reconfigure
```

**Option B — Manual Configuration**

Copy the example file and edit it manually:
```bash
cp .env.example .env
```

Edit `.env`:
```text
TELEGRAM_BOT_TOKEN=123456789:ABCDefghIJKlmnOPQrstUVwxYZ
TELEGRAM_USER_ID=123456789
```

> **Note:** Only the Users listed in `TELEGRAM_USER_ID` are authorized to use this bot. It silently ignores all requests from strangers.

### Step 3: Run the Bridge
Start up Antigram alongside your connected IDE or browser agent:

```bash
npm start
```

## 🛠️ Usage & Commands

Send these to your bot in Telegram!

### Agent Control
*   `[Any Text]` - Injects text prompt immediately into the IDE agent.
*   `🎤 Voice Note` - Injects free transcribed audio into the IDE.
*   `/stop` - Aborts the current agent generation task.
*   `/status` - Sends a snapshot screenshot of the immediate browser window state.

### File Management
*   `/list` - Show folders and files in active local workspace.
*   `/read <filename>` - Outputs file content as markup in Telegram.
*   `/download <filename>` - Downloads the file layout physically to your device.
*   `[File/Photo Upload]` - Automatically saves the item safely to the active local workspace.

### Environment Control
*   `/terminal <command>` - Runs a command locally and returns terminal output.
*   `/logs` - Previews the most recent terminal logs natively in Telegram.

### Git Version Control
*   `/diff` - Display `git status -s`.
*   `/commit <message>` - Add all and locally commit. 
*   `/push` - Push successfully committed code to the repository.
